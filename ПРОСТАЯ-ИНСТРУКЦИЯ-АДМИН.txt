ШАГ 1: Откройте SQL скрипт
───────────────────────────────────────────────────────────────

1. В Supabase Dashboard откройте SQL Editor
2. Откройте файл: supabase/migrations/003_fix_admin.sql
   (он в папке проекта на вашем компьютере)
3. Скопируйте весь код

ШАГ 2: Замените email
───────────────────────────────────────────────────────────────

1. Вставьте код в SQL Editor в Supabase
2. Найдите строку: 'ВАШ_EMAIL@example.com'
3. Замените на ваш реальный email
   Например: 'imrealstanyxx@gmail.com'
   Или: 'rokanoname007@gmail.com'
4. ВАЖНО: оставьте кавычки!

ШАГ 3: Выполните
───────────────────────────────────────────────────────────────

1. Нажмите Run (зелёная кнопка)
2. В результатах должна появиться таблица
3. В таблице должен быть ваш email и role = 'admin'

ШАГ 4: Проверьте
───────────────────────────────────────────────────────────────

1. Выйдите из аккаунта на сайте (если залогинены)
2. Войдите снова с вашим email и паролем
3. Откройте "Профиль"
4. Должно быть написано: "Администратор (полный доступ)"
