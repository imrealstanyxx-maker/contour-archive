ИНСТРУКЦИЯ: КАК ДОБАВИТЬ GITHUB ТОКЕН БЕЗОПАСНО

⚠️ ВАЖНО: GitHub токен - это секретная информация!
Нельзя коммитить токен в публичный репозиторий - GitHub автоматически заблокирует push.

РЕШЕНИЕ:

1. Откройте файл assets/config.js ЛОКАЛЬНО (на вашем компьютере)

2. Найдите строку:
   GITHUB_TOKEN: 'YOUR_GITHUB_TOKEN_HERE'

3. Замените на ваш токен:
   GITHUB_TOKEN: 'ваш_токен_здесь'

4. Сохраните файл

5. НЕ КОММИТЬТЕ этот файл в Git! 
   Токен должен остаться только на вашем компьютере.

6. Для работы на GitHub Pages:
   - Вариант А: Сделайте репозиторий приватным (Settings -> Change visibility)
   - Вариант Б: Используйте GitHub Secrets + GitHub Actions (сложнее)
   - Вариант В: Используйте Fine-grained token с минимальными правами только на issues

АЛЬТЕРНАТИВНОЕ РЕШЕНИЕ (для публичного репозитория):

Создайте Fine-grained Personal Access Token с минимальными правами:
1. GitHub -> Settings -> Developer settings -> Personal access tokens -> Fine-grained tokens
2. Generate new token
3. Выберите только ваш репозиторий
4. Permissions: Issues -> Read and write
5. Это менее опасно, если токен утечет, но всё равно лучше не коммитить

ВАШ ТОКЕН (добавьте его локально в assets/config.js):
[Вставьте ваш токен здесь локально, но НЕ коммитьте файл!]
