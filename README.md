# КОНТУР — Архив контурных единиц

Публичный архив контурных единиц (неофициальная компиляция)

## Описание

Архив материалов, относящихся к так называемым контурным единицам. Источники разрознены, происхождение части документов не установлено, целостность данных не гарантируется.

## Структура

- `index.html` — главная страница архива
- `dossier.html` — просмотр досье
- `partial.html` — неполная компиляция
- `terminal.html` — терминальный интерфейс
- `internal-access.html` — активация внутреннего доступа
- `about.html` — информация об архиве

## Доступ

- **Публичный** — материалы доступны всем
- **Утечка** — материалы, полученные неофициальным путём
- **Внутренний** — материалы системы КОНТУР (требуется код доступа)

## Технические детали

Статический сайт на HTML/CSS/JavaScript. Данные хранятся в `assets/data.js`. Работает без сервера.
